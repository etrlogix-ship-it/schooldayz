# 04. Hangman Word Game ⭐

## 📖 What You'll Build
The classic Hangman game! Guess letters to reveal a hidden word before you run out of lives.

## 🛒 Parts List
- Raspberry Pi (any model)\n- No extra parts needed!

## ⏱️ Time Needed
About 30–60 minutes

## 🔌 Wiring / Setup
No wiring needed.

## 💻 The Code

See `code/hangman.py` for the full program.

To run it:
```bash
python3 code/hangman.py
```
