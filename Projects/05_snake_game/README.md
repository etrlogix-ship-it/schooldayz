# 05. Terminal Snake Game ⭐⭐

## 📖 What You'll Build
A snake game that runs in the terminal using curses! Control the snake with arrow keys and eat the food to grow.

## 🛒 Parts List
- Raspberry Pi (any model)
- No extra parts needed!

## ⏱️ Time Needed
About 45 minutes

## 🔌 Wiring / Setup
No wiring needed. Install curses (usually pre-installed):
```bash
sudo apt install python3-dev
```

## 💻 The Code
```bash
python3 code/snake_game.py
```

## 🎉 What to Try Next
- Increase the speed
- Add obstacles
- Keep a high score!
