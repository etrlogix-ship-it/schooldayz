# 09. Typing Speed Test ⭐

## 📖 What You'll Build
Test how fast you can type! The Pi shows you a sentence, you type it as fast as you can, and it tells you your WPM (words per minute).

## 🛒 Parts List
- Raspberry Pi (any model)
- No extra parts needed!

## 💻 The Code
```bash
python3 code/typing_speed.py
```
