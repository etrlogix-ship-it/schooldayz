# 17. ASCII Photo Converter ⭐⭐

## 📖 What You'll Build
Turn any image into ASCII art using characters! A classic nerdy effect.

## 🛒 Parts List
- Raspberry Pi
- An image file (JPG or PNG)

## 🔌 Setup
```bash
pip3 install Pillow --break-system-packages
```

## 💻 The Code
```bash
python3 code/ascii_photo.py myimage.jpg
```
