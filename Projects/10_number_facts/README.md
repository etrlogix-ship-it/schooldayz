# 10. Number Facts Machine ⭐

## 📖 What You'll Build
Enter any number and the Pi will tell you fascinating facts about it — is it prime? Even or odd? A perfect square? And more!

## 🛒 Parts List
- Raspberry Pi (any model)
- No extra parts needed!

## 💻 The Code
```bash
python3 code/number_facts.py
```
