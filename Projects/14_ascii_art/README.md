# 14. ASCII Art Generator ⭐

## 📖 What You'll Build
Turn your name or any text into giant ASCII art using the terminal! Great for personalising your Pi setup.

## 🛒 Parts List
- Raspberry Pi (any model)
- pyfiglet library

## 🔌 Setup
```bash
pip3 install pyfiglet --break-system-packages
```

## 💻 The Code
```bash
python3 code/ascii_art.py
```
