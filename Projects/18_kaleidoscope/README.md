# 18. Kaleidoscope Generator ⭐⭐

## 📖 What You'll Build
Create stunning kaleidoscope patterns on screen using Python's turtle library!

- Raspberry Pi with a screen
- No extra parts needed!

## 🔌 Setup
```bash
sudo apt install python3-tk
```

## 💻 The Code
```bash
python3 code/kaleidoscope.py
```
