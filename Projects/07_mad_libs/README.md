# 07. Mad Libs Story Generator ⭐

## 📖 What You'll Build
A funny story generator! You fill in random words (nouns, verbs, adjectives) without knowing the story, then laugh at the silly result.

## 🛒 Parts List
- Raspberry Pi (any model)
- No extra parts needed!

## ⏱️ Time Needed
About 20 minutes

## 🔌 Wiring / Setup
No wiring needed.

## 💻 The Code
```bash
python3 code/mad_libs.py
```

## 🎉 What to Try Next
- Write your own stories
- Add more story templates
- Read the result out loud!
