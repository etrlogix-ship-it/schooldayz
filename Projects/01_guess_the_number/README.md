# 01. Guess the Number Game ⭐

## 📖 What You'll Build
A classic number guessing game in the terminal. The Pi picks a secret number and you try to guess it!

## 🛒 Parts List
- Raspberry Pi (any model)\n- No extra parts needed!

## ⏱️ Time Needed
About 30–60 minutes

## 🔌 Wiring / Setup
No wiring needed — this is a pure Python project!

## 💻 The Code

See `code/guess_the_number.py` for the full program.

To run it:
```bash
python3 code/guess_the_number.py
```
