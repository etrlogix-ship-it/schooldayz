# 100 🏆 AI Quiz Master

**Category:** 🧠 AI & Cool Tech  
**Difficulty:** ⭐⭐ Intermediate  
**Time:** 20 minutes

## What You'll Make
The ultimate project — an AI-powered quiz game that generates *brand new questions* on any topic you choose! Unlike the quiz from Project 3, this quiz is different every single time because the questions are made up fresh by an AI.

## What You'll Learn
- AI-powered content generation
- JSON parsing from AI responses
- Building polished interactive games
- How to combine everything you've learned!

## Parts You'll Need
Just your Raspberry Pi and internet connection!

## Setup

```bash
pip install anthropic
export ANTHROPIC_API_KEY="your-key-here"
python3 quiz_ai.py
```

## How It Works
1. You choose a topic (e.g. "space", "dinosaurs", "football")
2. You choose difficulty (easy/medium/hard)
3. The AI generates 5 brand new questions just for you
4. You answer each question
5. See your score!

Every game is different! The AI never repeats the exact same questions. Ask about "dolphins" one day and "roller coasters" the next — it knows about almost everything!
