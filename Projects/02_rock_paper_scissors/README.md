# 02. Rock Paper Scissors ⭐

## 📖 What You'll Build
Play Rock, Paper, Scissors against the Raspberry Pi! Type your choice and see who wins.

## 🛒 Parts List
- Raspberry Pi (any model)\n- No extra parts needed!

## ⏱️ Time Needed
About 30–60 minutes

## 🔌 Wiring / Setup
No wiring needed.

## 💻 The Code

See `code/rock_paper_scissors.py` for the full program.

To run it:
```bash
python3 code/rock_paper_scissors.py
```
