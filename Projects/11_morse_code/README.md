# 11. Morse Code Translator ⭐

## 📖 What You'll Build
Type any message and the Pi will translate it to Morse code! You can also decode Morse code back to English.

## 🛒 Parts List
- Raspberry Pi (any model)
- No extra parts needed (add an LED for extra fun!)

## 💻 The Code
```bash
python3 code/morse_code.py
```
