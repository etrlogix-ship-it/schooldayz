# 16. Turtle Drawing Shapes ⭐⭐

## 📖 What You'll Build
Use Python's turtle module to draw colourful shapes and patterns on screen. Watch the turtle move in real time!

## 🛒 Parts List
- Raspberry Pi with a screen
- No extra parts needed!

## 🔌 Setup
```bash
sudo apt install python3-tk
```

## 💻 The Code
```bash
python3 code/turtle_art.py
```
