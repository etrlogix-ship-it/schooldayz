# 🍓 100 Kid-Friendly Raspberry Pi Projects

Welcome to the ultimate collection of Raspberry Pi projects for kids! Whether you're 8 or 18, a complete beginner or already comfortable with code, there's something here for you.

## 📚 How to Use This Collection

Each project lives in its own folder and includes:
- `README.md` — Full step-by-step guide
- `code/` — All the Python (or other) code you need
- `wiring_diagram.md` — How to connect any electronics (where needed)
- `parts_list.md` — What you need to buy or find

## 🛠️ Before You Start

1. Make sure your Raspberry Pi has Raspberry Pi OS installed
2. Open a terminal and run: `sudo apt update && sudo apt upgrade`
3. Install Python packages as described in each project

## 🆘 Getting Help

- Ask a parent, teacher, or older sibling
- Visit [raspberrypi.com](https://www.raspberrypi.com)
- Check the Raspberry Pi forums

Happy Making! 🎉
