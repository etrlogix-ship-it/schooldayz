# 🍓 100 Kid-Friendly Raspberry Pi Projects

A collection of 100 fun, beginner-friendly Python projects for the Raspberry Pi. Whether you want to build games, control LEDs, make music, or chat with a real AI — there's something here for everyone.

All projects work on any Raspberry Pi model. Most also run on a regular laptop in simulation mode, so you can try them before you have the hardware.

## 🚀 How to Use This Collection

**Option 1 — Download the zip**
Click the green Code button, then Download ZIP. Unzip it anywhere on your Pi.

**Option 2 — Clone with Git**
```bash
git clone https://github.com/YOUR-USERNAME/raspberry-pi-kids-projects.git
```

**Then pick a project and run it:**
```bash
cd 01_guess_the_number/code
python3 guess_the_number.py
```

Each project folder has its own README with full instructions, a parts list, and wiring diagrams where needed.

---

## 🛠️ What You'll Need

- A Raspberry Pi (any model) — or just a laptop to start
- Python 3 (already installed on Raspberry Pi OS)
- Internet connection (for the AI projects, 96–100)
- Some basic electronics for projects 86–95 (LEDs, buttons, resistors — all cheap and easy to find)

Most projects install their own libraries with a simple `pip install` command — each README explains exactly what to run.

## 🌱 Where to Start

**Total beginner?** Start with project 01 — Guess the Number. It's just 30 lines of Python and teaches you the basics.

**Want to try electronics first?** Jump to project 86 — LED Blinker. It's the "Hello World" of physical computing.

**Interested in AI?** Go straight to project 96 — AI Chatbot. It works in demo mode with no API key needed.

**Want a challenge?** Try project 74 — Neural Network from Scratch, or project 67 — Maze Solver.
